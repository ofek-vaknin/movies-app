# 🎬 Movies App

A responsive **Movies browsing app** built with **React + Vite**.
Includes filtering, search, pagination, and persistent likes using **LocalStorage**.

---

🔗 https://movies-app.vercel.app

## ✨ Features

* 🔎 Search movies by title
* 🎭 Filter by genre (Radio buttons)
* ⭐ Filter by rating (1–5 stars)
* ❤️ Like / Unlike movies (**saved in LocalStorage**)
* 📄 Pagination (Prev / Next)
* 📱 Responsive layout (Mobile + Desktop)

---

## 📦 Getting Started (Run Locally)

### 1️⃣ Clone the repository

```bash
git clone https://github.com/ofek-vaknin/movies-app.git
```

### 2️⃣ Navigate to project

```bash
cd movies-app
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm run dev
```

The app will run on:

```
http://localhost:5173
```
