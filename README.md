# 🎬 Movies App

A responsive **Movies browsing app** built with **React + Vite**.
Includes filtering, search, pagination, and persistent likes using **LocalStorage**.

## ✨ Features

* 🔎 Search movies by title
* 🎭 Filter by genre (Radio buttons)
* ⭐ Filter by rating (1–5 stars)
* ❤️ Like / Unlike movies (**saved in LocalStorage**)
* 📄 Pagination (Prev / Next)
* 📱 Responsive layout (Mobile + Desktop)

---

* **React**
